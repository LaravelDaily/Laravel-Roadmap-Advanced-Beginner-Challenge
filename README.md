## Installation

Setting up your development environment on your local machine :

- git clone https://github.com/sodaler/Laravel-Roadmap-Advanced-Beginner-Challenge.git
- cp .env.example .env
- composer install
- npm install
- crm:install
- php artisan serve
- npm run dev
